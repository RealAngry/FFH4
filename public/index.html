API Information

This is the official API server for HMPS School Management System. The API provides endpoints for managing students, users, authentication, and other school-related functions.

Our API follows REST principles and returns responses in JSON format.

Available Endpoints

  • GET /api/test - Test endpoint to verify API functionality
  • GET /api/status - Check the current API status
  • POST /api/auth/login - User authentication
  • GET /api/students - Get all students

Access Control

This API requires authentication for most endpoints. Please contact the system administrator for API access credentials.
